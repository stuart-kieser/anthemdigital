Building Management Features

Building Overview

Access a comprehensive overview of all your buildings. This feature provides essential information about each building, helping you stay organized and informed about your property portfolio.

Building List

View a detailed list of all your buildings. This feature allows you to see all your properties in one place, making it easy to manage and access information about each building.

Building Table

Get a tabular view of your buildings. This feature organizes your properties into a table format, providing a clear and concise way to view and compare key details of each building.

Building Dashboard

Monitor key metrics for your buildings with a comprehensive dashboard. This feature provides insights into the total number of buildings, average rent, and total rent collected, helping you keep track of your financial performance.

Specific Building Dashboard

View detailed metrics for a specific building. This feature gives you insights into the number of rooms, vacancies, tenants, average rent, and total rent for a selected building, allowing for focused management of individual properties.

Add Building Extensions

Access specific forms for different types of buildings. This feature provides tailored forms for adding commercial, residential, or storage buildings, ensuring that all relevant information is captured accurately.

Submit New Building

Submit details of a new building and its associated documents. This feature allows you to upload documents, ensuring that all necessary paperwork is stored and linked to the relevant building record.

Building Card

View detailed information about a specific building. This feature provides a card view with essential details about a selected building, offering a quick and easy way to access key information.

Building Info Expanded

Access expanded information about a building. This feature provides an in-depth view of a building's details, including all associated data and documents, giving you a comprehensive understanding of the property.
