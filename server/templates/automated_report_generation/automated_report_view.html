Reporting Management Features

Reports Overview

Access a centralized hub for managing all reporting activities. This feature provides a comprehensive overview, allowing you to create, view, and manage reports efficiently.

Report Toolbox

Access a suite of tools designed for efficient report management. This feature includes various utilities to streamline report creation and organization.

Past Reports

View a list of all previously generated reports. This feature allows you to review and manage historical reports, ensuring you can access past data whenever needed.

Create Reports

Generate new reports based on selected buildings and report types. This feature allows you to specify the details and types of reports needed, including property, financial, asset, and tenant reports.

Files Table

View a table of documents associated with a specific building. This feature displays all files related to a building, helping you manage and organize building documents efficiently.

Add File

Upload and attach files to a specific building. This feature allows you to select and upload multiple documents, ensuring all relevant files are stored and linked to the appropriate building.

Delete File

Remove a document from the database. This feature allows you to delete files that are no longer needed, ensuring your document list remains accurate and up-to-date.

View File

Open and view the contents of a document. This feature provides an interface for viewing the selected file, ensuring you can easily access and review document contents.

File Storage

Access the storage directory for all uploaded files. This feature provides a path to the storage location, allowing you to retrieve and manage files directly from the storage directory.
