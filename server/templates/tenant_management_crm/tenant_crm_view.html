Room Management Features

Room List

This feature provides a detailed overview, allowing you to manage and organize rooms efficiently.

Room Table

View a detailed table of rooms filtered by building. This feature ensures that you can quickly check room statuses, including vacancy and pricing, for efficient management.

Add Room Form

Add new rooms to the building with ease. This feature provides a form to input necessary details such as room number, area, rate, and occupants, tailored to different building types.

Room Card

View detailed information about a specific room. This feature provides a comprehensive card layout showing all the key details of a room, making it easy to review room-specific information.

Change Payment Status

Update the payment status of tenants. This feature allows you to mark payments as paid and record the payment date, ensuring accurate tracking of tenant payments.

Room Options

Fetch and display options for rooms within a specific building. This feature provides a list of rooms filtered by building, making it easy to manage room-specific tasks and assignments.

Tenant Management Features

Tenant View

Access a general overview of tenant management. This feature provides a user-specific view, displaying essential information and quick access to tenant-related functionalities.

Tenant List

View a list of all tenants. This feature helps in managing and organizing tenant information effectively, ensuring easy access to tenant details.

Tenant Table

Display a detailed table of tenants. This feature includes information about tenant rooms and their respective prices, providing a comprehensive view for efficient management.

Specific Tenant Table

View tenants of a specific building. This feature filters tenants by building, allowing for a focused view of tenants in a particular building.

Tenant Card

View detailed information about a specific tenant. This feature provides a card layout displaying all key details of a tenant, making it easy to review tenant-specific information.

Tenant Dashboard

Access a dashboard with tenant-related statistics. This feature provides insights into the number of leases, unpaid tenants, average rent, total revenue, and unpaid revenue, helping in effective management.

Search Tenants

Search for tenants based on various criteria. This feature allows you to quickly find tenants using keywords related to their name, ID, room number, or payment status.

Payment History

View the payment history of a specific tenant. This feature provides a detailed record of all payments made by the tenant, ensuring transparent and accurate tracking.

View Payment

Display details of a specific payment. This feature shows the payment amount, date, and status, helping in reviewing individual payments.
